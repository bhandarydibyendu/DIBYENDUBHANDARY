# SOUMIK MUKHERJEE - Portfolio Website

A modern, responsive portfolio website showcasing my skills, experience, and projects as a Full Stack Software Developer.

## 🚀 Features

- Responsive design optimized for all devices
- Modern UI with smooth animations using Framer Motion
- Interactive tech stack showcase
- Project portfolio with detailed descriptions
- Contact form for inquiries
- Dark theme with gradient accents

## 🛠️ Tech Stack

- React.js
- Tailwind CSS
- Framer Motion
- Font Awesome Icons
- React Scroll
- Spring Boot (Backend)
- MySQL (Database)

## 📱 Sections

- Hero Section with animated tech stack
- About Me
- Experience Timeline
- Projects Portfolio
- Contact Form
- Footer with Social Links

## 🎨 Design

- Modern dark theme
- Gradient accents
- Smooth animations
- Responsive layout
- Interactive elements

## 🔧 Installation

1. Clone the repository:
```bash
git clone https://github.com/mukherjeesoumik/SOUMIKMUKHERJEE.git
```

2. Install dependencies:
```bash
cd SOUMIKMUKHERJEE
npm install
```

3. Run the development server:
```bash
npm run dev
```
